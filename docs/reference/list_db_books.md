# List the names of the codebooks in the database

List the names of the codebooks in the database

A wrapper for list_db_books to maintain backwards compatibility

## Usage

``` r
list_db_books(db_folder)

list.db_books(db_folder)
```

## Arguments

- db_folder:

  A string path to the database controlled vocabulary folder
