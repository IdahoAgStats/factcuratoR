# Assure that var_id is the same for the current outputfuzzymatch_df and the aux_fuzzy_status that is read in

Assure that var_id is the same for the current outputfuzzymatch_df and
the aux_fuzzy_status that is read in

## Usage

``` r
assure_var_id(output_fuzzymatch_df, aux_fuzzy_status)
```

## Arguments

- output_fuzzymatch_df:

  A data.frame with the newly generated fuzzymatches

- aux_fuzzy_status:

  A data.frame containing the fuzzymatches with is_truematch manually
  checked
