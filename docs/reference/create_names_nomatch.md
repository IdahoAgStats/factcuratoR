# Create a data.frame of the names of varieties with no match

Create a data.frame of the names of varieties with no match

## Usage

``` r
create_names_nomatch(df_nomatch, ...)
```

## Arguments

- df_nomatch:

  A data.frame of varieties without cultivar matches from the database.
  This will generally be the output of the function clean_manualfuzzy().

- ...:

  Extra columns to select
