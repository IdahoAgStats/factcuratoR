Check for overlap in strings This function takes a dataframe and
compares two columns based on whether the strings overlap. TRUE is
returned if there is an overlap in either direction (it doesn't matter
which string is longer)

## Usage

``` r
is_string_overlap(df, x1, x2)
```

## Arguments

- df:

  A data.frame

- x1:

  The bare variable name of the column to compare

- x2:

  The bare variable name of the column to compare
